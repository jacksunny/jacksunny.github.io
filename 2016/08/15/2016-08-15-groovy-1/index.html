安装

下载传送门中提供了三种安装方式
sdkman、homebrew，macPorts，我用的 homebrew，没有遇到困难

intellij idea 安装 groovy 环境

首先先下载intellij idea，安装，下一步下一步什么的都略过，最后安装好了，此时是无法正确常见一个 groovy 项目的，需要以下几步

1. 确认groovy插件是否已经安装

‘preferences –> plugin’ 中输入groovy 查看是否已经安装groovy插件，如果没有安装，点击该页面 Browse repositories 按钮 搜索并安装

2. 正确配置groovy library

安装插件后 点击 ‘ File –> New –> Project ‘ 找到 Groovy 选项卡,下图红框中应该对应之前安装的 groovy的某个版本，如果没有，点击三角，如果有提示，选中即可，如果 没有提示，即需要点击箭头所指按钮，手动加入 groovy library。

  • 如果是 homebrew 安装，则 文件路径: /usr/local/Cellar/groovy/2.4.7/libexec
  • 如果是其他方式安装，请自行google

HelloGroovy运行

创建 Groovy 工程后，在src 目录上右键选择 Groovy Script 即可

右键 创建的 Groovy Script 文件，选择 run，即可运行并在控制台输出结果
